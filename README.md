# bank-customer-churn-analysis-powerbi
Bank Customer Churn Analysis Dashboard
1. Project Title / Headline
A dynamic and interactive Power BI dashboard designed to analyze and visualize customer churn behavior in the banking sector—focusing on customer demographics, account activity, and financial attributes.

2. Short Description / Purpose
The Bank Customer Churn Dashboard helps identify patterns and key factors influencing customer attrition. It enables data-driven decision-making for improving customer retention and business strategies.

3. Tech Stack
The dashboard was built using the following tools and technologies:
• 📊 Power BI Desktop – Main platform for creating interactive dashboards
• 📂 Power Query – Used for data cleaning and transformation
• 🧠 DAX (Data Analysis Expressions) – For calculated measures and KPIs
• 📝 Data Modeling – Established relationships between customer attributes
• 📁 File Formats – .pbix (development) and .png (preview image)

4. Data Source
Source: Kaggle
The dataset contains bank customer information including:
Customer demographics (age, gender, geography)
Account details (balance, salary, credit card status)
Activity status (active/inactive customers)
Churn status (Exited or Retained customers)

5. Features / Highlights

• Business Problem
Customer churn is a major challenge in the banking industry. Retaining existing customers is more cost-effective than acquiring new ones, but identifying churn patterns from raw data is difficult.

Key questions include:
Which customers are more likely to leave?
Does geography or gender impact churn?
How do balance, age, and activity influence customer retention?
• Goal of the Dashboard
To create an interactive tool that:

Analyzes customer churn trends
Helps identify high-risk customer segments
Supports strategic decision-making for retention
• Walkthrough of Key Visuals

🔹 KPI Cards (Top Section)
Average Age, Balance, Credit Score
Total Customers
Total Exited Customers
Active Customers
Average Salary

🔹 Gender Filter & Country Filter
Interactive slicers to analyze churn by gender (Male/Female) and geography (France, Germany, Spain).

🔹 Churn by Geography (Map Visualization)
Shows geographic distribution of exited customers.

🔹 Exited by Gender (Donut Chart)
Displays churn proportion between male and female customers.

🔹 Age Distribution vs Churn (Histogram)
Highlights which age groups have higher churn rates.

🔹 Customer Count by Churn (Pie Chart)
Shows ratio of retained vs exited customers.

🔹 Balance vs Churn (Bar Chart)
Analyzes how account balance impacts churn probability.

• Business Impact & Insights
📉 Churn Reduction: Identify high-risk customers early
🎯 Targeted Marketing: Focus on specific demographics prone to churn
💡 Better Decision-Making: Use insights to improve customer retention strategies
🌍 Regional Insights: Understand churn trends across different countries
